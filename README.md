🚀 Quick Start
1. Clone & Install

bash
git clone https://github.com/sengobasar/AI-ROADMAP-GENERATOR.git
cd AI-ROADMAP-GENERATOR
npm install
2. Environment Setup
Configure:

text
GEMINI_API_KEY=your_gemini_api_key
MISTRAL_API_KEY=your_mistral_api_key
3. Run Locally

bash
npm start
# Visit http://localhost:3001
